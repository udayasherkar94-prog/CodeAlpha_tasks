# CodeAlpha Frontend Development Tasks

This repository contains the tasks completed during the **CodeAlpha Virtual Internship Program** for **Frontend Development**.

## 🚀 Technologies Used

* HTML5
* CSS3
* JavaScript

## 📂 Tasks

### Task 1 – Image Gallery

A responsive **Image Gallery** built using HTML, CSS, and JavaScript where users can view images in a clean grid layout.

### Task 2 – Calculator

A simple **Calculator** built using HTML, CSS, and JavaScript that performs basic arithmetic operations like addition, subtraction, multiplication, and division.

### Task 3 – Portfolio Website

A personal **Portfolio Website** to showcase projects, skills, and contact information using HTML, CSS, and JavaScript.

### Task 4 – Music Player

A **Music Player Web Application** built using JavaScript that allows users to play, pause, and control audio tracks.
